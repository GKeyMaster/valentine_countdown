# Building Textures (CC0)

**Manual step required:** Download 3 CC0 textures and place them here:

| File | Source | Description |
|------|--------|-------------|
| `facade_01.webp` | [Poly Haven](https://polyhaven.com/textures) or [ambientCG](https://ambientcg.com/) | Facade texture (concrete tiles, wall cladding, etc.) |
| `facade_02.webp` | Same | Alternate facade texture |
| `roof_01.webp` | Same | Roof tiles texture |

**Requirements:**
- CC0 licensed (commercial use, redistribution allowed)
- Export as WEBP at 1024px or 2048px
- Color/albedo only (no normals/roughness needed for this prototype)

**Example sources:**
- Poly Haven: search "concrete facade" or "rectangular tiles"
- ambientCG: search "roof tiles" or "exterior wall"
